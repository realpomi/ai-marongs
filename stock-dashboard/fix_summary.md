일일 주가 변화(캔들)가 표시되지 않는 문제를 수정했습니다.

**원인 분석 및 수정 내용:**
1.  **데이터 전달 방식 개선**: 서버에서 DB 데이터를 가져올 때 객체 속성이 명확히 전달되도록 코드를 수정했습니다. 기존 방식(Spread 연산자 사용)이 특정 DB 라이브러리 객체와 호환되지 않아 데이터가 누락되었을 가능성을 차단했습니다.
2.  **데이터 정합성 강화**:
    - 차트 라이브러리(`lightweight-charts`)가 요구하는 엄격한 **시간 오름차순 정렬**을 적용했습니다.
    - **중복 날짜 제거** 로직을 추가하여, 데이터 중복으로 인한 차트 렌더링 실패를 방지했습니다.
    - 유효하지 않은 데이터(숫자가 아닌 값 등)를 필터링하는 안전장치를 추가했습니다.

**확인 방법:**
페이지를 새로고침하여 차트가 정상적으로 표시되는지 확인해 주세요.
만약 여전히 차트가 보이지 않는다면, 브라우저 개발자 도구(F12)의 콘솔(Console) 탭에 붉은색 에러 메시지나 'No valid chart data available' 경고가 있는지 확인 부탁드립니다.
